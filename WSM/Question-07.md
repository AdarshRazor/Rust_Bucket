## Question 
Spektra Systems is a large enterprise that has been operating for many years with extensive on-premises infrastructure. They have a significant investment in their on-premises servers and software licenses, including SQL Server. Recently, the company has decided to migrate all of its on-premises infrastructure to Microsoft Azure. What would allow them to perform this migration with minimal licensing costs.

## Options 
Option 1 : Azure Hybrid Benefit
Option 2 : Azure Virtual Machines
Option 3 : Azure Reservations
Option 4 : Azure Functions
Option 5 : Azure Discounted Pricing

## Answers 
Option 1 : 12 

## Reference Links 
https://learn.microsoft.com/en-us/training/modules/m365-azure-migrate-replicate-virtual-servers/replicate-virtual-machines  

## Explanation 
Leveraging Azure Hybrid Benefit would be highly beneficial for Spektra Systems as it allows them to minimize licensing costs while migrating their existing on-premises SQL Server workloads to Azure.
